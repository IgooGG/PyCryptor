# PyCryptor
![Pylint](https://github.com/IgooGG/PyCryptor/actions/workflows/pylint.yml/badge.svg)  
A python encryptor for python using AES-256  
To install use:  
    python3 -m pip install -r requirements.txt  
and then  
    python3 encrypt.py  
This should open a encryption menu.  
If something went wrong during installation install on linux python-tk
